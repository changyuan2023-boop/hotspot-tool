**⚠️ 严禁幻觉——每一条必须有原文出处：**
- 每条输出内容（包括事件描述、股价涨跌幅、具体数字、财报数据）必须能在下方「资讯来源」中找到对应原文，否则直接删除该条，不得保留
- 若资讯来源仅提供标题、没有具体数字，则**不得补充任何数字**，标注「[数据待确认]」
- 若某主题在资讯来源中完全没有出现（包括「资讯来源」末尾标注为 0 站出现的主题），**禁止生成该主题的任何条目**

第一行输出标题，固定为（直接复制，不修改任何字）：

**条目排列顺序（从上到下）：**

3. 其他板块与个股（能源、医药、消费等）
4. 其他值得关注的消息

要求：
- **全文使用中文**，无论资讯来源是英文还是中文，一律用中文概括
- 股票代码（如 NVDA、TSLA）和无中文译名的品牌简称（如 SpaceX、OpenAI）可保留，其余一律中文
- 全文 **10 条以内，能少则少**，无来源的内容不写
- **同一主题的相关消息合并为一条**
- 每条 50-100 字为宜
- 不分级别，不写前言，不写结尾说明，直接输出

## 早间模式操作指令

聚焦：隔夜美股收盘结果 + 影响今日港股开盘的信息。

重点关注（按重要性排序写入）：
1. 美股三大指数昨夜涨跌及核心驱动，一句话概括（必须有资讯来源支撑，不得推断）
2. 个股重大异动（财报或重大消息驱动涨跌 5% 以上）
3. 隔夜重要宏观数据（非农/CPI/PCE/美联储表态——仅限资讯来源中明确出现）
4. 隔夜消息中可能影响港股相关板块的内容，重点关注：光通信、存储/HBM、半导体、中概互联网等；若有明确利好/利空驱动，须点明对应港股板块或个股

## 什么内容值得写入

值得写（必须有明确标的或板块，且资讯来源中有对应内容）：
- 美联储利率决议 / 主席讲话（**仅限资讯来源中明确提及**）
- 七大科技股（AAPL / MSFT / NVDA / GOOGL / META / AMZN / TSLA）财报或重大消息
- 个股异动 5% 以上 + 有明确原因
- 行业板块整体联动（多股同向）

不要写：
- 泛泛的市场情绪描述
- 没有标的的宏观背景铺垫
- 已持续多日的旧闻（除非有新进展）
- **任何在「资讯来源」中找不到原文依据的内容**
