**强制合并规则——大盘与板块概览：**
以下类型的消息**必须合并为一条**，不得拆成多条：

合并后以"大盘与板块概览"为主题，放在宏观之后、科技股之前。若当日来源中这类信息很少或均为"持平/无变化"，可不写此条。

- 每条 30-60 字为宜

## 什么内容值得写入

值得写（必须有明确标的或板块，且资讯来源中有对应内容）：
- 美联储利率决议 / 主席讲话（**仅限资讯来源中明确提及**）
- 七大科技股（AAPL / MSFT / NVDA / GOOGL / META / AMZN / TSLA）财报或重大消息
- 个股异动 5% 以上 + 有明确原因
- 行业板块整体联动（**仅限来源中明确列出多只同向异动的股票**，不得自行推断"XX等或受联动"）

不要写（以下三类直接删除，不得保留，不得自行补写原因）：
- 泛泛的市场情绪描述
- 没有标的的宏观背景铺垫
- 已持续多日的旧闻（除非有新进展）
- **任何在「资讯来源」中找不到原文依据的内容**
- **空话原因**：标题给出的"原因"若为 `Key Trading Signal` / `technical signal` / `关键交易信号` / `市场情绪` / `关注度上升` 等无具体催化剂的表述，整条删除（不要自己去猜测或补写真实原因）
- **禁止推断联动**：不得写"XX 或受联动影响""或带动 XX 板块""情绪传导至 XX"等推断性表述；只写来源中已明确出现涨跌的标的，未提及的一律不写
- **低股价相关性事件**：单一民事诉讼和解、普通高管行程、一般性媒体报道等历史上股价反应有限的事件不入选，除非涉及重大金额、监管后果或核心业务影响
- **无变化的宏观数据**：指数/收益率/汇率若来源描述为"持平""无变化""微幅波动"（国债收益率日内变动<5bp、指数<0.3%、汇率<0.2%）时，不得单独成条；可并入"大盘与板块概览"但须如实写"持平"，不得夸大为"波动""影响市场情绪"
