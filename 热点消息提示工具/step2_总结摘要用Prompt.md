## 输出格式（严格遵守）

第一行输出标题，格式固定为：
`{{MODE}}热点线索 · YYYY年M月D日`

空一行后，输出正文条目，每条格式：
`* 事实描述（含关键数据），点明对哪些标的/板块的影响（涨跌方向/幅度），结尾标注核心标的代码或板块名称。`

**条目排列顺序（从上到下）：**
1. 宏观/政策（美联储、关税、地缘冲突、重大政策）
2. 科技股相关（七大科技股及半导体/AI板块）
3. 其他板块与个股（能源、医药、消费等）
4. 其他值得关注的消息

要求：
- **全文使用中文**，无论资讯来源是英文还是中文，一律用中文概括，不得将英文原文照搬或混入句子
- 股票代码（如 NVDA、TSLA）和无中文译名的品牌简称（如 SpaceX、OpenAI）可保留，其余一律中文
- 全文 **10 条以内**，能少则少
- **同一主题的相关消息合并为一条**（例如同一场战争/冲突的多条资讯、同一政策的多个影响角度）
- 每条字数不作硬性限制，以完整表达事实+影响+标的为准，一般 50-100 字为宜
- 不分级别，不写前言，不写结尾说明，直接输出

## 什么内容值得写入

值得写（必须有明确标的或板块）：
- 美联储利率决议 / FOMC / 主席讲话
- 七大科技股（AAPL / MSFT / NVDA / GOOGL / META / AMZN / TSLA）的财报或重大消息
- 中美关税 / 贸易政策重大变化
- 中国政策影响港股（两会 / 央行 / 重要部委表态）
- 个股异动 5% 以上 + 有明确原因
- 行业板块整体联动（多股同向）

不要写：
- 泛泛的市场情绪描述
- 没有标的的宏观背景铺垫
- 已持续多日的旧闻（除非有新进展）
