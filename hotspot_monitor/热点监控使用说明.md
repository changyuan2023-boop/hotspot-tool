# 热点监控 · 使用说明

站内用户交易数据来自 Metabase 表（每 5 分钟更新），只能在你本机通过「复制表格」获取。本工具通过**一次粘贴 + 一次点击**，把站内交易人数与 LongPort 涨跌幅合并，输出美股、港股两张表，便于做热点监控。

## 流程简述

热点监控页**已内置格式转化**：直接粘贴 Metabase 复制的表格（含表头「标的ID」「市场」「实时成交人数」等）即可，无需单独打开转化页。

1. 在项目根目录执行 **`python3 hotspot_monitor/hotspot_monitor_api.py`** 启动后端（保持运行）；或进入 `hotspot_monitor` 目录执行 `python3 hotspot_monitor_api.py`（需在项目根或当前目录有 `.env`）。
2. 在浏览器打开 **http://127.0.0.1:5000/**（即热点监控页，由后端提供；不要用 `file://` 直接打开 HTML，否则会报 Failed to fetch）。
3. 在 Metabase 打开「实时成交榜-按成交人数」，选中表格（**含表头**）复制 → 回到热点监控页粘贴 → 点击「获取涨跌幅并生成榜单」。
4. 页面会展示两张表：**美股**、**港股**，均按**站内交易人数**排序，并附带**当前涨跌幅**和**最新价**。

## 前置条件

- **后端**：需先启动 API 服务（拉取 LongPort 涨跌幅）  
  - 在项目根目录执行：`python3 hotspot_monitor/hotspot_monitor_api.py`  
  - 确保 `.env` 中已配置 LongPort 凭证（`LONGPORT_APP_KEY`、`LONGPORT_APP_SECRET`、`LONGPORT_ACCESS_TOKEN`）。
- **前端**：用浏览器访问 **http://127.0.0.1:5000/**（后端会提供热点监控页）。不要用 `file://` 直接打开 `hotspot_monitor.html`，否则请求 API 会被浏览器拦截（Failed to fetch）。

## Metabase 与格式转化

- Metabase 表头通常包含：**标的名称**、**标的ID**（如 `ST/HK/9988`、`ETF/US/SOXL`）、**市场**、**实时成交人数** 等。复制时**必须含表头**。
- 热点监控页会**自动识别**该格式，将「标的ID + 市场」转为标的列：港股补成 5 位 + `.HK`（如 `09988.HK`），美股 + `.US`（如 `TSLA.US`），再拉取涨跌幅。若你粘贴的已是「标的 + 交易人数」的 .US/.HK 表格，也会正常解析。

## 如何简化日常使用

- **书签**：把 **http://127.0.0.1:5000/** 加到浏览器书签，启动后端后即可一键打开热点监控页。
- **快捷方式**：在 Metabase 把该报表固定到首页，需要时：打开报表 → 全选表格 → 复制 → 打开热点监控页 → 粘贴并点击「获取涨跌幅并生成榜单」。
- **数据更新**：Metabase 表每 5 分钟更新，你只需在需要看热点时复制最新表格并粘贴刷新即可；涨跌幅由本工具请求 LongPort 实时拉取。

## 排查「Failed to fetch」

- **先启动后端**：必须在终端运行 `python3 hotspot_monitor_api.py`（或从项目根 `python3 hotspot_monitor/hotspot_monitor_api.py`），保持不关闭。
- **用正确地址打开页面**：在浏览器地址栏输入 **http://127.0.0.1:5000/** 打开（由后端提供页面），不要双击 `hotspot_monitor.html` 用 `file://` 打开。
- **5000 端口被占用**：若启动时报 `Address already in use`（macOS 上常见因 AirPlay 占用 5000），可执行 `PORT=5001 python3 hotspot_monitor_api.py`，然后打开 **http://127.0.0.1:5001/**。
- 打开页面后若顶部显示「后端已连接 ✓」说明正常；显示「后端未连接」则按上面步骤检查。

## 文件说明

| 文件 | 说明 |
|------|------|
| `hotspot_monitor.html` | 热点监控页：粘贴 Metabase 或 .US/.HK 表格 → 自动转化（如需）并展示美股/港股两张表 |
| `hotspot_monitor_api.py` | 后端：`POST /api/quotes` 按 symbols 返回涨跌幅 |
| `.env` | LongPort 凭证（勿提交到 git） |
